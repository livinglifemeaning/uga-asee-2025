# ASEE Robotics 2025 Competition

## Overview
This repository contains the collaborative codebase for our team’s entry in the **ASEE Robotics 2025 Competition**. The goal is to program an autonomous robot that can move across the playing field, collect the foam golf balls, sort the golf balls by color, and deposit the golf balls into three bins based on the color of the ball. 

## Team Collaboration
All team members will use this GitHub repository to:
- Contribute code for robot control, sensors, and navigation.
- Track issues, bugs, and feature requests.
- Review and merge pull requests.
